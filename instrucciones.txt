- Para que funcione la libreria texttable, ya que esto me sirve para crear la tabla de los vértices y distancias que se muestran 
	al utilizar el Algortimo de Bellman-Ford se debe colocar:
	pip install texttable

- Para agregar vértices del grafo se debe colocar: 
	agregar vertice 0
	agregar vertice 1
	agregar vertice 2
		
- Para agregar aristas, este es del tipo: agregar arista(vertice origen, vertice destino, peso)
	se debe colocar:
	agregar arista 0 1 2
	agregar arista 1 2 3
	
- Para utilizar el Algoritmo de Bellman-Ford, este es del tipo: bellman-ford(vertice origen)
	Desde el vértice origen 1, se debe colocar:
	bellman-ford 1

- Para que se muestre los vértices y las aristas
	se debe colocar:
	mostrar

- Para salir del grafo 
	se debe colocar:
	salir
